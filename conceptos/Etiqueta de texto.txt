Etiqueta de texto 

-- <h1> </h1>

Nombre: Heading 1 / Encabezado de nivel 1

Función: Define el título principal de una página o sección.

Sintaxis: <h1>Texto del encabezado</h1>

Ejemplo: <h1>Bienvenidos a mi página web</h1>

-- <h2> </h2>

Nombre: Heading 2 / Encabezado de nivel 2

Función: Define un subtítulo de segundo nivel.

Sintaxis: <h2>Texto del encabezado</h2>

Ejemplo: <h2>Información personal</h2>

-- <h3> </h3>

Nombre: Heading 3 / Encabezado de nivel 3

Función: Define un subtítulo de tercer nivel.

Sintaxis: <h3>Texto del encabezado</h3>

Ejemplo: <h3>Datos de contacto</h3>

-- <h4> </h4>

Nombre: Heading 4 / Encabezado de nivel 4

Función: Define un subtítulo de cuarto nivel.

Sintaxis: <h4>Texto del encabezado</h4>

Ejemplo: <h4>Dirección</h4>

-- <h5> </h5>

Nombre: Heading 5 / Encabezado de nivel 5

Función: Define un subtítulo de quinto nivel.

Sintaxis: <h5>Texto del encabezado</h5>

Ejemplo: <h5>Referencias</h5>

-- <h6> </h6>

Nombre: Heading 6 / Encabezado de nivel 6

Función: Define el encabezado de menor importancia.

Sintaxis:<h6>Texto del encabezado</h6>

Ejemplo: <h6>Nota adicional</h6>

-- <p> </p>

Nombre: Paragraph / Párrafo

Función: Permite escribir bloques de texto o párrafos.

Sintaxis: <p>Texto del párrafo</p>

Ejemplo: <p>Este es un párrafo de ejemplo en HTML.</p>

-- <b> </b>

Nombre: Bold / Negrita

Función: Muestra el texto en negritas.

Sintaxis: <b>Texto en negrita</b>

Ejemplo: <b>Importante</b>

-- <u> </u>

Nombre: Underline / Subrayado

Función: Subraya el texto contenido entre las etiquetas.

Sintaxis: <u>Texto subrayado</u>

Ejemplo: <u>Texto destacado</u>

-- <i> </i>

Nombre: Italic / Cursiva

Función: Muestra el texto en cursiva o itálica.

Sintaxis: <i>Texto en cursiva</i>

Ejemplo: <i>Énfasis en una palabra</i>

-- <br>

Nombre: Line Break / Salto de línea

Función: Inserta un salto de línea dentro del texto sin crear un nuevo párrafo.

Sintaxis: <br>

Ejemplo: <br>
